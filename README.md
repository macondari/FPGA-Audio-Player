# FPGA-Audio-Player

WAV audio player implemented on an FPGA with SD card support and audio output.

## Features
- Reads WAV files from SD card using FAT filesystem.
- Plays stereo audio via FPGA audio interface.
- Playback controls via buttons: play/pause, stop, next, previous.
- Displays track info and status on an LCD.
- Adjustable playback speed modes controlled by hardware switches.
- Interrupt-driven button and timer handling for responsive control.
